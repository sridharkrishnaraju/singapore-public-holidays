# Singapore Public Holidays

A Singapore public holidays widget: the gazetted holidays for 2026 and 2027, with the day of the week and observed-Monday dates. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="sg-holidays.js"></script>
<sg-holidays></sg-holidays>
```

## What it does

Singapore has 11 gazetted public holiday days a year. When one falls on a Sunday, the following Monday is a public holiday — which is how the calendar picks up several long weekends. The dates follow the Ministry of Manpower (MOM) gazette.

## When you'd use it

Planning leave, scheduling deliveries or campaigns, or showing an accurate holiday calendar on a Singapore website.
