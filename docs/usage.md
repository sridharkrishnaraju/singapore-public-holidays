# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="sg-holidays.js"></script>
<sg-holidays></sg-holidays>
```

## React

```jsx
import "@sgbp/sg-holidays";
export default function Page() { return <sg-holidays />; }
```

## Vue

```vue
<script setup>
import "@sgbp/sg-holidays";
</script>

<template>
  <sg-holidays />
</template>
```
