# Singapore Public Holidays

A Singapore public holidays widget: the gazetted holidays for 2026 and 2027, with the day of the week and observed-Monday dates. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/singapore-public-holidays)**

```html
<script src="sg-holidays.js"></script>
<sg-holidays></sg-holidays>
```

## What it does

Singapore has 11 gazetted public holiday days a year. When one falls on a Sunday, the following Monday is a public holiday — which is how the calendar picks up several long weekends. The dates follow the Ministry of Manpower (MOM) gazette.

## Install

```bash
npm install @sgbp/sg-holidays
```

or copy `sg-holidays.js` into your project.
